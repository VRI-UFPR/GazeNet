### The official repository for VRI-GazeNet

![Gif demo](/ferris.gif)
## Installation

* Install required packages:
```
pip install -r requirements.txt  
```

## Demo

*  Run:
```
 python photo_cpu \
 --snapshot models/VRI.pkl \
 --gpu cpu \
 --image {Image Path} \
```
This means the demo will run using *VRI.pkl* pretrained model
